# Secure Password Generator Using Python

## Description

This project is a Secure Password Generator built using Python. It generates strong and random passwords to improve account security.

## Features

- Generate strong random passwords
- User-defined password length
- Uses letters, numbers, and special characters
- Fast and easy to use
- Improves password security

## Technologies Used

- Python
- Random Module
- String Module

## How to Run

1. Open terminal or command prompt.
2. Navigate to the project folder.
3. Run:

```bash
python password_generator.py
